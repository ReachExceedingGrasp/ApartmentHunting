***********************************************************************************************
Apartment Hunting v1.00
***********************************************************************************************

This project is intended for student who are from out of state and need to find housing neaer their respective universities. As an international student myself, I found this task quite daunting and hope this project helps out other students and makes their life a little easier.

***********************************************************************************************
Getting Started
***********************************************************************************************
Go to direcctory where the ApartmentHunting.class file is and run it using moj_run.

***********************************************************************************************
Prerequisites
***********************************************************************************************

Web Browser (Any)
